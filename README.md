# file-chunk-upload

## Installation
```sh
npm install file-chunk-upload
```
### Import
- ESM
```js
import FileUploader from 'file-chunk-upload'
```
- CommonJS
```js
var FileUploader = require('file-chunk-upload')
```
- CDN

```html
<script src="https://unpkg.com/file-chunk-upload/dist/file-chunk-upload.min.js"></script>
```

## Example
```sh
npm install
npm run dev
```
